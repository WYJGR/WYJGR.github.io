Abstract

Underwater Monocular Depth Estimation (UMDE) is a critical task that aims to estimate high-precision depth maps from underwater degraded images caused by light absorption and scattering effects in marine environments. Recently prevailing Mamba-based methods have achieved the promising performance across various vision tasks, however, these approaches perform poorly in the UMDE task since their inflexible state scanning strategies fail in modeling structural features of underwater images. Meanwhile, existing UMDE datasets usually contain unreliable depth labels, leading to incorrect object-depth relationships between underwater images and corresponding depth maps. To overcome these limitations, we develop a novel tree-aware Mamba method, dubbed as Tree-Mamba, for estimating accurate monocular depth maps from underwater degraded images. Specifically, a tree-aware scanning strategy is proposed to adaptively construct minimum spanning tree based on feature similarity, and spatial topological features among tree nodes are flexibly aggregated through bottom-up and top-down traversals, enabling to achieve stronger capabilities of multi-scale feature representation. Moreover, we construct an Underwater Depth Estimation Benchmark (UDEB), including 38,162 underwater image pairs with reliable depth labels, as a baseline for training existing deep learning-based UMDE methods to learn accurate object-depth relationships. Extensive experiments demonstrate the superiority of the proposed Tree-Mamba over several leading methods in both qualitative results and quantitative evaluations with competitive computational efficiency.

Underwater Depth Estimation Benchmark (UDEB)

The schematic diagram of constructing our UDBE dataset. Dataset A contains all underwater images from the existing seven UMDE datasets (Sea-Thru, NYU-U, SQUID, FLSea, Atlantis, SUIM-SDA, and USOD10K), while Dataset B stores all pseudo-labeled depth maps for each underwater image from Dataset A. Dataset C is a subset randomly selected from Dataset B, while Dataset D is its subjectively labeled counterpart. The generation and selection of high-quality depth maps includes four main steps: Step 1: six state-of-the-art MDE methods (AdaBins, DPT, Marigold, Depth Anything V1, Depth Anything V2, and Lotus Depth) are used to generate pseudo-labeled depth maps from Dataset A. Step 2: the depth maps are subjectively evaluated from Dataset B. Step 3: the LAR-IQA model is fine-tuned with Dataset D. Step 4: the fine-tuned LAR-IQA model is employed to automatically select the best depth maps from Dataset B.

Tree-aware Mamba (Tree-Mamba)

The overview of our proposed Tree-Mamba architecture. An underwater image first passes the CNN-based encoder to produce multi-scale feature maps. Then, these feature maps are fed into Conv 3×3 layers for reducing feature channel. Next, Mamba fusion layers are used to refine these feature maps, where the proposed tree-aware scanning strategy adaptively constructs minimum spanning tree with spatial topology structure from input feature maps and propagates multi-scale structure features by tree state propagation. The depth map is finally produced through the prediction head.
